# Change-controller-domain-CCD
highly specialized program, for server admins.
This application transfer roles, such as:
* PDCEmulator
* RIDMAster
* InfrastructureMaster
* SchemaMaster
* DomainNamingMaster.

Based on ntdsutil.

## The module i used ##
1. PySide2
2. Subprocces
3. Threading
4. socket
5. pythonping

### Perfectly work on Windows Server 2012B2, maybe another. ###
